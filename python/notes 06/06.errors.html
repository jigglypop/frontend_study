      # 예외 처리 
      ​
      ## 기본  - `try` `except`
      `try` 구문을 이용하여 예외 처리를 할 수 있습니다.
      ​
      기본은 다음과 같은 구조를 가지고 있습니다.
      ​
      ```python
      try:
          codeblock1
      except 예외:
          codeblock2
      ```
      ​
      * `try`절이 실행됩니다. 
      ​
      * 예외가 발생되지 않으면, `except`없이 실행이 종료가 됩니다.
      ​
      * 예외가 중간에 발생하면, 남은 부분을 수행하지 않고, `except`가 실행됩니다.

      4  예외 처리¶

      4.1  기본 - try except¶

      try 구문을 이용하여 예외 처리를 할 수 있습니다.

      기본은 다음과 같은 구조를 가지고 있습니다.

      try:
    codeblock1
except 예외:
    codeblock2

      • try절이 실행됩니다.

      • 예외가 발생되지 않으면, except없이 실행이 종료가 됩니다.

      • 예외가 중간에 발생하면, 남은 부분을 수행하지 않고, except가 실행됩니다.

      ## 복수의 예외 처리
      ​
      * 두 가지 예외를 모두 처리할 수 있습니다.

      try:
          codeblock1

          codeblock2

          print('숫자가 맞는지 확인좀;')
      except ZeroDivisionError:
          print('0으로 나눌수는 없지....')
      except:
          print('무슨 일인지는 모르겠지만 오류야')

      # `assert`
      ​
      `assert` 문은 예외를 발생시키는 다른 방법이다. 
      ​
      보통 상태를 검증하는데 사용되며 무조건 `AssertionError`가 발생한다.
      ​
      ```
      assert Boolean expression, error message
      ```
      ​
      위의 검증식이 거짓일 경우를 발생한다.
      ​
      `raise`는 항상 예외를 발생시키고, 지정한 예외가 발생한다는 점에서 다르다.

      6  assert¶

      assert 문은 예외를 발생시키는 다른 방법이다.

      보통 상태를 검증하는데 사용되며 무조건 AssertionError가 발생한다.

      assert Boolean expression, error message

      위의 검증식이 거짓일 경우를 발생한다.

      raise는 항상 예외를 발생시키고, 지정한 예외가 발생한다는 점에서 다르다.
